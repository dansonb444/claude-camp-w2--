# Claude Camp W2 练习说明

本目录包含 4 个 Python 小程序：词频统计（图形界面）、用户管理器、待办清单（JSON 持久化）、安全计算器。以下说明各自功能、运行方式，并各举一例。

---

## 1. 词频统计器 `word_frequency.py`

**功能**：在窗口中粘贴英文文本，统计单词出现次数（忽略大小写），用字典保存结果并按次数从高到低显示；另有单词查询框，可按词匹配查询是否出现及次数。

**运行**（需图形界面环境）：

```bash
cd claude-camp-w2--
python3 word_frequency.py
```

**示例**：在绿色边框的大文本框中输入 `Hello world hello`，点击「统计词频」，结果中会看到 `hello` 出现 2 次、`world` 出现 1 次；在「要查的单词」框输入 `world`，点「查询」，会提示包含该词及次数。

---

## 2. 用户管理器 `user_manager.py`

**功能**：用字典按邮箱索引存储用户（姓名、邮箱、加入日期）；命令行菜单支持添加、查询、删除；预置 10 条示例数据；输入异常时提示，不崩溃。

**运行**：

```bash
cd claude-camp-w2--
python3 user_manager.py
```

**示例**：启动后选 `2`（查询），提示输入邮箱时输入 `all`，可列出全部预置用户；再选 `2`，输入 `chen.siyuan@example.com`，可看到「陈思远」的完整信息。

---

## 3. 待办清单 `todo_list.py`

**功能**：添加待办、按编号标记完成、查看清单；未完成与已完成前用不同符号（如黄色圆点 / 绿色对勾）；数据读写同目录下的 `todos.json`，重启后自动加载；若启动时该文件不存在会报错并退出（需先准备合法 JSON 文件，例如内容为 `[]`）。

**运行**：

```bash
cd claude-camp-w2--
python3 todo_list.py
```

**示例**：选 `1` 添加一条「买牛奶」，选 `3` 查看清单，选 `2` 输入编号 `1` 将其标为完成，再选 `3` 可看到该项前为已完成图标。关闭程序后再次运行，清单会从 `todos.json` 恢复。

---

## 4. 安全计算器 `safe_calculator.py`

**功能**：支持 `+`、`-`、`×`、`÷`（以及 `*`、`/`、`x` 等等价写法）；除数为 0 或非数字输入时只提示、不崩溃；输入 `quit` 优雅退出。

**运行**：

```bash
cd claude-camp-w2--
python3 safe_calculator.py
```

**示例**：在 `>` 后输入 `10 ÷ 2`，得到 `= 5.0`；输入 `5 / 0`，提示除数不能为 0；输入 `abc + 1`，提示格式或数字不合法；输入 `quit` 结束程序。

---

## 环境说明

- 上述命令均使用 **Python 3**。
- `word_frequency.py` 依赖系统自带的 **Tk**；在 macOS 上若使用系统 Python，可能出现 Tk 相关提示，可按脚本内说明处理。
